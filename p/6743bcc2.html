关于前端布局的一些知识

2023/4/22 笔记 前端

这篇笔记将简单介绍一下前端布局，包括其优缺点，以及其大致的应用场景。布局的选择在页面主体结构开发中至关重要，选择合适的布局模式，在开发过程中，会和德芙一样，纵享丝滑～

css的发展路线

  • CSS1.0：这是CSS的第一个版本，于1996年底发布，主要定义了一些基本的样式属性，如字体、颜色、背景、边距、对齐等，以及一些基本的选择器，如元素选择器、类选择器、ID选择器等。CSS1.0的目的是实现内容和表现的分离，让网页更加美观和易于维护。
  • CSS2.0：这是CSS的第二个版本，于1998年5月发布，基于CSS1.0的基础上增加了很多高级的功能，如浮动和定位、高级选择器、伪类和伪元素、媒体类型、表格样式等。CSS2.0的目的是提供更多的布局和交互能力，让网页更加灵活和丰富。
  • CSS3.0：这是CSS的第三个版本，目前还没有最终定稿，但已经有很多模块被浏览器支持。CSS3.0是基于模块化开发的，每个模块都有自己的规范和状态。CSS3.0增加了很多强大的新功能，如动画和过渡、变形和转换、弹性盒子和网格布局、圆角和阴影、渐变和滤镜、字体和图标等。CSS3.0的目的是提供更多的创意和效果，让网页更加美观和动感。

从 1.0 我们只能进行简单的样式更改, 到 2.0 可以更方便的处理 表格 媒体 更易于的对元素选择. 再到 3.0 我们可以对样式进行更加丰富, 具有想象力的创作. 我们接触了大部分的样式操作，但对过于复杂的页面样式，大家在写项目时，也应该感受到了布局的选择局限了我们的开发效率，为调整布局结构而浪费大量时间。

下面部分将介绍一下，css发展至今的5大布局。

五大布局

固定布局

固定布局是一种网页布局方式，它的特点是网页内容被一个固定宽度的容器包裹，容器内的区块都有固定的百分比或者像素宽度值。1 固定布局的优点是能够精确控制网页的设计和排版，不受浏览器窗口大小的影响。2 固定布局的缺点是不能充分利用浏览器的空间，可能会出现空白或者滚动条，不适合移动端和响应式设计。2

固定布局通常使用CSS的position属性来设置元素的定位类型，其中有以下几种常用的值：

  • static：默认值，没有定位，元素出现在正常的文档流中。
  • relative：相对定位，元素相对于自身原来的位置进行偏移，但不影响文档流中其他元素的位置。
  • absolute：绝对定位，元素脱离文档流，相对于最近的已定位的祖先元素进行偏移，如果没有则相对于body元素。
  • fixed：固定定位，元素脱离文档流，相对于浏览器窗口进行偏移，不随滚动条而移动。
  • inherit：从父元素继承position属性的值。

固定布局还需要使用left、right、top或者bottom来设置元素相对于参照元素的偏移值。

从上面的具体介绍，我们也知道固定布局便是我们最初接触的传统布局形式，有点时代的眼泪的感觉，我们在使用的时候也会发现，他并不适合作为页面的主体布局模式。在这里就不过多介绍了。

流式布局

流式布局是一种网页布局方式，它的特点是页面中的元素的宽度按照屏幕分辨率自动进行适配调整。这样可以保证在不同的设备上，页面中的元素大小都能适应屏幕的变化。流式布局也叫百分比布局，因为它主要使用百分比来设置元素的宽度。

  • 下面是一个简单的例子，使用流式布局来实现一个三列的布局，左右两列固定宽度，中间一列自适应。
<div class="container">
  <div class="left">左侧</div>
  <div class="middle">中间</div>
  <div class="right">右侧</div>
</div>

.container {
  width: 100%;
}

.left {
  width: 200px;
  float: left;
}

.right {
  width: 200px;
  float: right;
}

.middle {
  margin-left: 200px;
  margin-right: 200px;
}

流式布局可以任意改变窗口大小、比例，但容器中的文字，无法做出相应的改变。

弹性布局

弹性布局是一种CSS3的布局模式，它可以让容器中的子元素在任何方向上排列，并且可以自动调整它们的大小，以适应不同的屏幕和设备¹²。弹性布局也叫flex布局或flexbox布局，因为它主要使用flex属性来控制子元素的排列和伸缩³⁴。弹性布局是一种更灵活和高效的布局方式，可以替代传统的block和inline布局⁵。

下面是一个简单的示例，使用弹性布局来实现一个三列布局，左右两列固定宽度，中间一列自适应。

<div class="container">
  <div class="left">左侧</div>
  <div class="middle">中间</div>
  <div class="right">右侧</div>
</div>

.container {
  display: flex; /* 设置容器为弹性布局 */
}

.left {
  width: 200px; /* 设置左侧列固定宽度 */
}

.right {
  width: 200px; /* 设置右侧列固定宽度 */
}

.middle {
  flex: 1; /* 设置中间列自适应 */
}

自适应布局

自适应布局是一种网页设计的方法，它可以让网页的内容根据不同的设备屏幕分辨率而进行适应¹³⁵。自适应布局的特点是分别为不同的屏幕分辨率定义布局，即创建多个静态布局，每个静态布局对应一个屏幕分辨率范围³⁴。自适应布局可以使用CSS媒体查询（media query）来实现¹⁵。自适应布局与响应式布局的区别是，响应式布局是一种流式布局，它可以让网页的内容随着浏览器窗口的大小而动态调整¹⁵。

好的，我根据你的需求写了一个自适应布局的代码，你可以看看效果是否符合你的预期。我使用了 CSS Grid 来实现栅格布局，你可以在这里了解更多关于 CSS Grid 的知识：https://developer.mozilla.org/zh-CN/docs/Web/CSS/CSS_Grid_Layout

这是我的代码：

<html>
<head>
<style>
.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px; /* 左右两列固定宽度，中间一列自适应宽度 */
  grid-gap: 10px; /* 每列之间有10px的间隔 */
}

.item {
  height: 100px;
  background-color: lightblue;
}

@media only screen and (max-width: 600px) { /* 当屏幕宽度小于600px时 */
  .container {
    grid-template-columns: 1fr; /* 变成一列布局 */
  }
}
</style>
</head>
<body>
<div class="container">
  <div class="item">左侧栏</div>
  <div class="item">中间栏</div>
  <div class="item">右侧栏</div>
</div>
</body>
</html>

响应式布局

你好，这是Bing。根据我的搜索结果，响应式布局指的是同一页面在不同屏幕尺寸下有不同的布局¹⁴⁵。响应式布局的目的是让网页能够适应不同的设备和浏览器，提高用户体验和兼容性⁴。响应式布局有几种方法，常见的有以下几种¹²³：

  • 使用 **CSS 媒体查询（media query）**来根据不同的屏幕宽度设置不同的样式规则。
  • 使用百分比、rem、em 等相对单位来设置元素的宽度、高度、字体大小等，使之能够随着父元素或根元素的变化而变化。
  • 使用 flexbox 或 grid 布局来实现弹性的栅格布局，让元素能够自动填充或缩放空间。
  • 使用 viewport meta 标签来控制网页在移动设备上的缩放比例和视口宽度。
  • 使用图片或字体的响应式加载技术，根据设备的分辨率和网络状况来加载不同质量或大小的资源。

好的，我根据你的需求写了一个响应式布局的代码，你可以看看效果是否符合你的预期。我使用了 flex 布局来实现弹性的栅格布局，你可以在这里了解更多关于 flex 布局的知识：https://developer.mozilla.org/zh-CN/docs/Web/CSS/CSS_Flexible_Box_Layout

这是我的代码：

<html>
<head>
<style>
.container {
  display: flex; /* 使用 flex 布局 */
  flex-wrap: wrap; /* 允许元素换行 */
}

.item {
  height: 100px;
  margin: 5px; /* 每个元素之间有 10px 的间隔 */
}

.item-1 {
  background-color: lightblue;
  flex: 0 0 200px; /* 左侧元素固定宽度为 200px */
}

.item-2 {
  background-color: lightgreen;
  flex: 1 1 auto; /* 中间元素自适应宽度 */
}

.item-3 {
  background-color: lightpink;
  flex: 0 0 200px; /* 右侧元素固定宽度为 200px */
}

@media only screen and (max-width: 600px) { /* 当屏幕宽度小于600px时 */
  .item {
    flex-basis: 100%; /* 所有元素占满一行 */
  }
}
</style>
</head>
<body>
<div class="container">
  <div class="item item-1">左侧栏</div>
  <div class="item item-2">中间栏</div>
  <div class="item item-3">右侧栏</div>
</div>
</body>
</html>

看完上面部分我们可以发现，响应式布局和自适应布局实现的关键点，便是==css媒体查询==

/* 自适应 */

  }
  
/* 响应式 */

单位

单位 作用 特性
px 页面按精确像素展示 绝对单位
rem 相对根节点html字体大小来计算 相对单位
em 基准点为相对父节点字体大小 相对单位
% 相对元素的大小设定的比例 相对单位
vw 视图的宽度的百分比，1vw代表视图宽度的1% 视图单位
vh 视图的高度的百分比，1vh代表视图高度的1% 视图单位
vmin 当钱vw和vh中较小的一个值 视图单位
vmax 当前vw和vh中较大的一个值 视图单位
